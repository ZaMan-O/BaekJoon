### 출력 

 <p>첫 번째 줄에 시작점과 도착점을 포함한 로봇의 이동 경로 위에서 자기장이 최대가 되는 지점의 좌표를 공백으로 구분하여 출력하라. 만약, 가능한 답이 여러 개라면 아무거나 출력해도 된다. 또한, 자기장이 최대가 되는 모든 지점의 좌표는 정수임을 증명할 수 있다.</p>

<p>만약, 로봇의 이동 경로 중 시작점과 도착점을 포함해 로봇이 이동 중 도선의 위치를 지난다면 대신 <span style="color:#e74c3c;"><code>-1</code></span>을 출력하라.</p>
